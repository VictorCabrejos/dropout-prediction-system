# Diagramas de Patrones de Diseño

## Estructura Actual vs. Estructura Propuesta

```
ESTRUCTURA ACTUAL:
┌───────────┐       ┌─────────────────┐       ┌─────────────────┐
│   API     │─────▶│ DropoutPredictor │─────▶│ LogisticRegression │
└───────────┘       └─────────────────┘       └─────────────────┘
```

```
ESTRUCTURA PROPUESTA:
┌───────────┐       ┌────────────────┐       ┌───────────────┐       ┌────────────┐
│   API     │─────▶│ PredictionFacade │─────▶│ Decorators    │─────▶│ Predictor  │
└───────────┘       └────────────────┘       └───────────────┘       └────────────┘
                           │                                               │
                           │                                               ▼
                           │                  ┌───────────────┐       ┌────────────┐
                           └─────────────────▶│ InputAdapter  │       │ ModelFactory│
                                              └───────────────┘       └────────────┘
```
